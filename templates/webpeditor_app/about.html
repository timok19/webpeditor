WebP Editor

What this application is about?

This is a modern and ease to use web application to edit images in limited formats and also allows to convert images in some specific formats. All in one place.

Is this application is free to use?

Yes, it is 100% free to use. No payments whatsoever.

What kind of information can I get from uploaded picture/image?

After uploading image, you will be able to see specification of the image on Info page. Those infos are file name, file size, resolution, format, color mode, aspect ratio and even further you will see an EXIF data of the image (if exists).

How many files can I upload?

Maximum amount of files to be uploaded is 15 with specific restrictions. Individual file should have max. 6 MB and sum of all file size should not overflow 50 MB.

Can I change quality of the output image on convert?

Yes, you can do it. On Converter page under format input, you can choose between minimum and maximum quality from 5 to 100 points. Depending on your choice, file size and quality will be increased or decreased.

What is image editor is capable of?

WebP Editor can provide basic image editing with popular instruments such as cropper, rotation, resizing, adding objects such as text, geometric shapes. Also, there is some filters that you can apply on the image. For more information just upload your first image.
